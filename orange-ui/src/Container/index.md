# Container 按钮

用于布局的容器组件，方便快速搭建页面的基本结构：

- `<or-container>`:外层容器。
- `<or-header>`:顶栏容器。
- `<or-aside>`:侧边栏容器。
- `<or-main>`:内容容器。
- `<or-footer>`:底栏容器
